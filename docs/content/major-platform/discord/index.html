Hands-On Exercises (OSINT + CTF)

1) Find & Validate a Server Invite

  1. Google: site:discord.gg "<your keyword>" to find an invite.
  2. Open invite; note server name, member count, and vanity URL (if any).
  3. In-app, use in:#welcome + has:link to find onboarding info.

2) Timeline via Snowflake IDs

  1. Copy a message link â†’ extract the Snowflake ID.
  2. Convert with Snow-Stamp or Unfurl.

  3. Validate scope/permission before opening anything.

5) Bot Surface Mapping

  2. List permissions, features, and linked dashboards/APIs.

  3. Use hashes in queries ("SHA256") to locate mirrors.

  2. Cross-check domains with Discord AntiScam.
  3. Document indicators: new accounts, typos, shortened URLs.

10) Evidence Hygiene & Chain

  1. Capture URLs, timestamps (Snowflake), and context notes.
  2. Store hashes of files (if in-scope) and screenshots with time.
  3. Maintain a small who/what/when/where/how ledger.
