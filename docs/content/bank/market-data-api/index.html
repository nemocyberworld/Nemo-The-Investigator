Usage notes
  • Most APIs require registering for a free API key.
  • Use stock tickers (e.g., AAPL, TSLA) or crypto symbols (BTC, ETH) as queries.
  • Check API limits — free tiers usually have daily/monthly caps.
  • Some APIs (like Quandl, OANDA) include commodities & macroeconomic data too.
