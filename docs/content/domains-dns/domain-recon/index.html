Technique notes
  • Pivot from domain → subdomains → hosting IPs → co-hosted domains.
  • Review SSL certificates (SAN fields) for hidden or related hostnames.
  • Check historical DNS and WHOIS changes to track ownership shifts.
  • Leverage Wayback Machine to recover old endpoints and tech stack details.
