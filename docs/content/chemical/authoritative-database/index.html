Usage notes
  • Use compound names, CAS numbers, InChI keys, or drug IDs for precise searches.
  • Databases differ in focus: reactions (Reaxys), literature (SciFinder), bioactivity (ChEMBL), virtual screening (ZINC), metabolism (HMDB), drugs (DrugBank), protein-ligand structures (PDBe).
  • Cross-reference across multiple databases for stronger validation.
