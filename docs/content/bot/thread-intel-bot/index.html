Technique notes
  • Start with reputation/blacklists → pivot to sandbox & relationships (VirusTotal/Hybrid/ANY.RUN) → expand via infra scans (Shodan/Censys) → verify sightings (SANS ISC/GreyNoise).
  • Normalize queries: strip protocol for domains, use SHA256 for samples when possible, and quote URLs in urlscan.
  • Correlate time windows: look for synchronized registrations, hosting reuse, and JA3/JA4 overlaps (SSLBL).
