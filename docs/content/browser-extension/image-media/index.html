Technique notes
  • Prefer original media URLs when possible (right-click → “Copy image address”).
  • Use multiple reverse-image engines (TinEye, RevEye, Search by Image) to cross-check.
  • Pull EXIF on originals; most platforms strip metadata on upload.
  • For video: capture reproducible timestamps; use frame-accurate tools for verification.
