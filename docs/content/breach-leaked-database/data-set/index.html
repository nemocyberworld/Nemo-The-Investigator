Technique notes
  • Start with meta-search (Google Dataset Search) → refine in domain portals (Data.gov, EU, World Bank) → jump to research repos (Kaggle/Zenodo/Dataverse).
  • Use filters: filetype:csv, parquet, json, or date ranges to narrow results.
  • Check dataset licenses and privacy terms before downloading or sharing; avoid handling unlawfully obtained PII.
