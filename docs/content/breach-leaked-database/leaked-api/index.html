Technique notes
  • Workflow: start with HIBP (email/domain) → paste/dump search (psbdmp, IntelX) → provider queries (BreachDirectory/Leak-Lookup/Scylla/DeHashed) → exposure context (EmailRep).
  • For passwords, don’t send the full hash. Use HIBP’s k-anonymity: SHA-1 the password locally, send the first 5 chars to /range/, then check suffixes client-side.
  • Respect rate limits, authentication, and data handling policies. Only query data you are allowed to access.
