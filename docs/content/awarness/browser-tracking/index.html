Awareness notes
  • Browser fingerprinting combines canvas, fonts, headers, and plugins to uniquely identify you.
  • Use privacy-hardened browsers (Firefox, Brave, LibreWolf) and extensions like uBlock, NoScript.
  • VPN/Tor only mask IP — they don’t fully stop fingerprinting.
